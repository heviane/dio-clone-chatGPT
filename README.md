# chatGPT-Clone-v1.0

chatGPT integration with Node and React.

## Technologies used in projects

### Back-end

- Javascript
- Node
- Express
- Cors

### Front-end

- Javascript
- React

### How to run the service from the backend?

```sh
$ cd server
$ npm install
$ npm start
```

The backend service must be running on port :5000.

### How to run the front-end service?

```sh
$ cd web
$ npm install
$ npm start
```
